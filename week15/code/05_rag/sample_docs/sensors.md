# Sensor Reference (excerpt)

## CO2 sensors
Conference rooms use CO2 sensors. A reading above 1,000 ppm triggers increased
fresh-air ventilation. A reading above 1,800 ppm in an empty room usually means
a calibration fault, not real occupancy.

## Temperature sensors
Each guest room has one wall-mounted temperature sensor. Readings are sampled
every 60 seconds and stored as events in the building knowledge graph.
