#   BRAIN=local   → a local OpenAI-compatible model (Ollama / vLLM on a DGX). Sovereign.

#   BRAIN=claude  → Anthropic Claude. needs `anthropic` + ANTHROPIC_API_KEY.
#   BRAIN=sim     → scripted stub, no model, $0 offline. needs nothing extra.
#   BRAIN=auto    → local if up, else claude, else sim (the default).
# The tripartite memory lives on local disk (.memory/) — it never leaves the box.
